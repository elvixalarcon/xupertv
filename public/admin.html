App Android — Actualizaciones

            

              

                Las películas no van dentro del APK: la app es un reproductor web que se conecta a tu servidor.
                Cuando cambias la web o el catálogo, los usuarios lo ven al instante. Solo hay que actualizar el APK cuando
                cambias código nativo (notificaciones, TV, etc.). Sube los APK a data/apk/ y sube el
                version code solo de la app que cambiaste (móvil y TV por separado).
              

              

              

              

                

                  
                  
                

                

                  
                  
                

                

                  
                  
                

                

                  
                  
                

                

                  
                  
                

                

                  
                

              

              

                
              

            

          

          

            
Proxy de streams — Ocultar IP del servidor

            

              

                Las señales de TV por Internet bloquean la IP del servidor. Activa proxies HTTP para resolver y reproducir
                con otra IP. Formato: una línea por proxy — http://usuario:clave@host:puerto
